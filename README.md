# Hyper-V WMI еҸӮиҖғ

`root\virtualization\v2` дёӢзҡ„ WMI зұ»гҖҒеұһжҖ§гҖҒж–№жі•еңЁеҗ„ Windows зүҲжң¬й—ҙзҡ„еҜ№з…§гҖӮ

## жөҸи§Ҳ

еңЁзәҝпјҡ<https://justsenger.github.io/HyperV-WMI-Documentation/>

жң¬ең°жңҚеҠЎеҷЁеҗҜеҠЁпјҡ

```bash
python -m http.server 8777
```

## ж•°жҚ®

еңЁ `data/db/`пјҡ

- `canonical.json` вҖ” дё»ж•°жҚ®гҖӮ
- `translations_zh.json` вҖ” зҝ»иҜ‘гҖӮ
- `examples.json` вҖ” зӨәдҫӢгҖӮ
- `glossary.json` вҖ” жңҜиҜӯиЎЁгҖӮ
